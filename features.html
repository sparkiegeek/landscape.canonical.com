Manage thousands of Ubuntu machines as easily as one

Landscape is the most cost-effective way to support and monitor large and growing networks of desktops, servers and clouds. Reduce your team's efforts required for basic, day-to-day management. Now featuring Autopilot, the fastest way to build an OpenStack cloud.

Powerful systems management

  • Manage desktop, server and cloud deployments
  • Up to 40,000 machines with a single instance
  • Create custom profiles for managing different classes of machines and ensure uniformity
  • Easily install, update, rollback and remove software
  • Define policies for automated updates and security patches
  • Configure users and groups

Monitor your machines at scale

  • Set alerts for updates on specific machines
  • Graph trends of temperature, disk, memory usage and system load
  • List all processes running on a system and remotely kill rogue processes
  • Build your own graphs with custom metrics

Maintain security and compliance

  • Patch compliance - keep systems secure and up to date
  • Role Based Access Control (RBAC)
  • Automated audit logging and compliance reporting
  • Regulatory compliance is significantly simplified with custom reporting

Learn more about compliance on ubuntu.com

Control your inventory

  • Quickly track full software package information for all registered machines
  • Gather asset information in real time
  • Create dynamic search groups to perform operations on categories of machines
  • Easily access any system’s hardware properties

Integrate Landscape with your current systems

  • Scriptable API to customise Landscape for your organisation’s needs
  • Integrate with your current configuration management, monitoring and ticketing systems
  • Automate and extend the way you use Nagios and Puppet

Package repository management

  • Mirror and stage internal or external APT repositories
  • Upload and manage custom packages

Autopilot is the easiest way to build an OpenStack cloud

  • Build and manage a highly resilient OpenStack cloud from the bare metal up
  • Autopilot accelerates installation and implements our best practice reference architecture based on available resources
  • The result is a cloud designed for both maintainability and expansion
  • Autopilot is included in Landscape Dedicated Server
  • You can try it on up to 10 machines for free
  • When you need to scale your cloud, you can buy extra Ubuntu Advantage licenses

Learn more about Autopilot on ubuntu.com
